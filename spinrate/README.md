# Spinrate 🎵

App de reseñas musicales. Para correrla localmente:

## Requisitos
- Node.js 18+

## Instalación

```bash
# 1. Descomprimí el zip y entrá a la carpeta
cd spinrate

# 2. Instalá dependencias
npm install

# 3. Corré en desarrollo
npm run dev
```

Luego abrí **http://localhost:3000** en el navegador.

## Variables de entorno
Ya vienen configuradas en `.env.local` con tus credenciales de Supabase.

## Deploy (cuando estés listo)
```bash
npm install -g vercel
vercel
```
